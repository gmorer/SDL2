# SDL2

## Dependencies

- SDL2
- SDL2\_ttf
- SDL2\_image

### Ubuntu

```
sudo apt update
sudo apt install libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev
```

## Installation

```
git clone https://github.com/gmorer/SDL2.git && cd SDL2 && make
```

## Usage

```
./line
```
