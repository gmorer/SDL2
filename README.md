# SDL2

## Dependencies

- SDL2
- SDL2\_ttf
- SDL2\_image

### Ubuntu

```
sudo apt update
sudo apt install libsdl2-dev sdl2_ttf sdl2_image
```

### ArchLinux

```
sudo pacman -S sdl2 sdl2_ttf sdl2_image
```

## Installation

```
git clone https://github.com/gmorer/SDL2.git && cd SDL2 && make
```

## Usage

```
./line
```
